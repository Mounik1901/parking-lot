# Parking Lot

=> How to Run

Following are the instructions to run the solution:

* run `gem install bundler` to install `bundler` gem in your environment

* run `bin/setup` to install needed gem listed in `Gemfile` file and run the unit test.

* run the `bin/parking_lot file_inputs.txt` in terminal to run the solution in file mode

or

* run `bin/parking_lot` (wihout input file name) to run solution in command mode
